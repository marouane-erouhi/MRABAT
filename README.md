# MRABAT

## build frontend
`cd client`

`npm install`

`npm run dev`


## run backend
make sure to include credetials in `.env` file

`cd server`

`npm install`

`npm run dev`


### `.env` file
```
MONGO_URI=MONGO_SERVER_LINK

# CLOUDINARY info here
CLOUDINARY_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_SECRET=

```
